# Castlebound: Siege Eternal

Castlebound: Siege Eternal is a top-down 2D medieval siege survival game built in Unity. Defend your castle from waves of enemies, repair walls, and survive as long as you can. Android-first, solo-dev.

## Feature Snapshot
- Wave-based siege survival loop
- Repairable barriers and castle defense focus
- Upgrade systems and progression (in development)

## Requirements
- Unity 2023 LTS (2D)

## Run the Game
1. Open the project in Unity 2023 LTS.
2. Open `Scenes/MainPrototype.unity`.
3. Press Play.

## Run Tests
- Unity Test Runner → EditMode or PlayMode → Run All

## Project Layout
- `Assets/_Project/Scenes` — main playable scenes
- `Assets/_Project/Scripts` — gameplay code
- `Assets/_Project/_Tests` — EditMode/PlayMode tests
- `Docs` — design and workflow docs
